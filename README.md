## 📖 About the Project

Secure Login System with OTP & CAPTCHA is a Flask-based web application designed to provide secure user authentication. The project includes user registration, login authentication, email OTP verification, CAPTCHA validation, password hashing, and an admin dashboard to enhance security.

## ✨ Features

- 🔐 Secure User Registration & Login
- 📧 Email OTP Verification
- 🤖 CAPTCHA Validation
- 🔑 Password Hashing
- 👨‍💼 Admin Dashboard

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Deshmukh-Yash/secure-login-system-with-otp-captcha.git
```

Go to project folder

```bash
cd secure-login-system-with-otp-captcha
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python app.py
```
